Pseudo-Classes & Pseudo-Elements Reference

Frequently Used Pseudo-Classes

Name Description Instance
:link Represents an element that has not yet been visited. a:link {color: red;}
:visited Represents links that the user has already visited. a:hover {color: orange;}
:hover Matches when the user interacts with an element with a pointing device, but does not necessarily activate it. a:hover {color: orange;}
:active Represents an element (such as a button) that is being activated by the user. a:active {color: red;}
:focus Represents an element (such as a form input) that has received focus. input:focus {color: red;}

Frequently Used Pseudo-Elements

Name Description Instance
:first-letter Applies styles to the first letter of the first line of a block-level element, but only when not preceded by other content (such as images or inline tables). p:first-letter {font-size: 130%;}
:first-line Applies styles to the first line of a block-level element. p:first-line {color: red;}
